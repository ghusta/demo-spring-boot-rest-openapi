== Getting started

=== Start database

```shell
docker run -d -p 5442:5432 ghusta/postgres-world-db:2.13
```

=== Start Spring Boot server

```shell
mvn spring-boot:run
```

== OpenAPI artefacts

=== Open Swagger UI

* http://localhost:8080/api-docs/ui
